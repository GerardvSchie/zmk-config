# Glove80 keyboard
This is a repository for the configuration of my glove80 keyboard taken from the [original-repo](https://github.com/moergo-sc/glove80-zmk-config). On a commit it automatically builds the new firmware for both sides of the keyboard.

## Quick start
Here are [instructions on how to firmware boot](https://docs.moergo.com/glove80-user-guide/customizing-key-layout/#putting-glove80-into-bootloader-for-firmware-loading) so the keyboard gets the new software. The [online editor](https://nickcoutsos.github.io/keymap-editor/) is a very good source to edit the config at.

## Docs
The original [moergo docs](https://docs.moergo.com/layout-editor-guide/) are located there, the site also contains a useful
[startup with suggestions](https://docs.moergo.com/glove80-user-guide/typing-with-glove80/#setting-up-glove80-for-comfortable-typing) many users have configured to have some simple features unique to programmable keyboard.

## Mods
Home row is where the keys where your fingers rest have additional behaviour, on long press they become GUI, Ctrl, Alt, Shift keys so you need to move less akwardly, here is [a guide on it](https://zmk.dev/docs/keymaps/behaviors/hold-tap).
